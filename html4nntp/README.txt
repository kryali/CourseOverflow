Html4Nntp is a web based newsgroup client written in PHP. It's
for the easy access of any NNTP server through your favourite web browser.
It's a fork of the projet Web-News: http://web-news.sourceforge.net/webnews.html

1. Requirements
- The scripts works for PHP 4.3.6 or later.
- Please make sure that the track_vars option is turned On
  (As of PHP 4.0.3, it's default value is On)

2. Configuration
After you have extracted all the files, you can configure html4nntp by editting
config/html4nntp.cfg.php file.
